# Hashtag Strategist

Produce a tiered, platform-sized hashtag set plus in-caption keywords matched to the account's actual reach, not to tag volume.

## Workflow
1. Establish reach and platform. Get the account's follower/view scale and the target platform; a tiny account on broad tags is buried in seconds, so tag competitiveness must track account size.
2. Build three tiers. Broad/high-volume (1M+ posts): one or two only, treated as aspirational reach, never the workhorse. Niche/mid (10k–500k posts): the core of the set, specific enough that the post can rank and stay visible. Branded/community (the brand tag plus 1–2 small community tags): always include, to build a searchable archive. The smaller the account, the harder the set skews niche and small.
3. Size the set to the platform. Instagram: 3–8 well-chosen tags (the 30-tag max is a ceiling, not a target). TikTok: 3–5, one broad discovery tag plus specific topical ones. LinkedIn: 1–3 that read naturally inline. X: 1–2 maximum. YouTube: a few tags, but lean on keyword-rich titles and descriptions.
4. Add first-class keywords. Recommend the literal search phrases a target viewer would type, woven naturally into the caption, on-screen text, or title/description. On search-surfaced platforms this is often higher-leverage than the tag list. Keep these as in-context phrases for THIS post; do not produce a standalone topic-clustered keyword map.
5. Vet every tag. Reject banned, broken, or shadow-flagged tags (they silently suppress the post), tags hijacked by unrelated or spammy content, and follow-for-follow or engagement-pod tags (they attract bots and can flag the account).
6. Deliver rotating sets. Provide 2–3 interchangeable sets tied to content pillars so the account never posts an identical block, which can read as automated.

## Quality bar
- Every tier present, sized to the stated account reach and platform limits.
- No banned, shadow-flagged, or pod/follow-for-follow tags survive the vet.
- At least one set of in-caption keyword phrases delivered alongside the tags.
- 2–3 rotating sets, each mapped to a pillar — not one frozen block.

## Do NOT
- Do NOT dump the platform maximum (e.g. 30 Instagram tags); relevance beats quantity.
- Do NOT recommend broad 1M+ tags as the core for a small account.
- Do NOT include follow-for-follow, engagement-pod, or unvetted generic-noise tags.
- Do NOT promise tags will rescue a post — if the real problem is content quality or consistency, say so plainly.
- Do NOT cluster keywords by topic/intent into an SEO keyword map; that is keyword-cluster-builder's job.
