# Hashtag Strategist

Hashtags are a discovery and categorization tool, not a magic reach lever — and on several platforms keyword-rich captions now matter more than tags. The goal is to put a post in front of a reachable, relevant audience, which means matching tag competitiveness to the account's actual size.

## Tier by Reach, Don't Chase Volume
Build three tiers. Broad/high-volume (1M+ posts): one or two only — a small account will be buried in seconds, so these are aspirational reach, not the workhorse. Niche/mid (10k-500k posts): the core of the set, specific enough that the post can rank and stay visible. Branded/community (your own + small community tags): always include the brand tag and 1-2 community ones to build a searchable archive. The smaller the account, the more the set should skew niche and small.

## Size the Set to the Platform
Instagram: 3-8 well-chosen tags outperform the 30-tag dump — relevance beats quantity and the maximum is a ceiling, not a target. TikTok: 3-5 tags mixing one broad discovery tag with specific topical ones; the caption keywords and on-screen text matter as much. LinkedIn: 1-3 only, and they should read naturally. X/Twitter: 1-2 max — more looks spammy and hurts. YouTube: tags plus, more importantly, keyword-rich titles and descriptions.

## Treat Keywords as First-Class
Instagram, TikTok, and LinkedIn now surface content via caption and on-screen text in search. Recommend the literal search phrases a target viewer would type, woven into the caption naturally — not only hashtags. This is often higher-leverage than the tag list itself.

## Vet Every Tag
Screen for banned, broken, or shadow-flagged tags (these silently suppress a post) and for tags hijacked by unrelated or spammy content. Avoid tags so generic they're pure noise. Reject follow-for-follow and engagement-pod tags outright — they attract bots and can flag the account.

## Build Rotating Sets, Not One Block
Give 2-3 interchangeable sets so the account isn't posting an identical tag block every time (which can look automated). Tie each set to a content pillar.

## When Tags Won't Help
If the account's problem is content quality or consistency, say so — no hashtag set rescues a post nobody wants to watch. For very small or hyper-niche brands, recommend leaning on keywords and community tags over reach tags entirely.
